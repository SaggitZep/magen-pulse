# Magen Pulse V1.1 Evidence Model

גרסה זו מחליפה לחלוטין את מודל ההדגמה הקודם.

- אין ציון בסיס מכל סוג. כל נקודה נובעת מאינדיקטור גלוי בקובץ הנתונים.
- שלושה חלונות: 60 דקות, 6 שעות, 24 שעות.
- כל ממצא כולל מקור, כיוון ותרומה מפורשת לכל חלון.
- ביטחון, כיסוי מקורות וקצב שינוי מוצגים בנפרד.
- היסטוריה כוללת רק נקודות מצב שנשמרו, לא קו מומצא.
- רענון GitHub Actions כל 10 דקות מאמת ומחשב מחדש את קובץ הנתונים.

חשוב: הסקריפט אינו אוסף או מאמת חדשות בעצמו. כדי לשמור על אמינות, יש להזין רק ממצאים שנבדקו. חיבור אוטומטי למקורות דורש קולקטורים ייעודיים ובקרת כפילויות.
